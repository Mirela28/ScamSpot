# ScamSpot

1. Project title
ScamSpot AI is an iOS application that helps users determine whether their case is a phishing attempt or not by using AI-assisted analysis. The app focues on case context analysis, clear explanations and confidence score guidance rather than a simple scam/not-scam output.

2. Overview
Many people experience digital doubt when encoutering suspicious messages or links
Existing apps analyse only one screenshot or text at a time with a fixed answer
ScamSpot allows users to input enough context, describe the case in detail and attach multiple files before analysis

3. Key features
- Upload suspicious screenshots
- Paste suspicious messages or URLs
- Explain your case in detail
- AI-based scam analysis
- Confidence score based on how much context was given
- Clear, non-technical explanations of outcome
- Feedback loop for improving future predictions
